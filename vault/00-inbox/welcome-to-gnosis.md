# Welcome to Gnosis

Gnosis is a personal knowledge base built on the [[Zettelkasten]] method. Every note is an atomic unit of thought, linked to other notes through [[Wikilinks]] so ideas can surface connections you didn't anticipate.

## How This Vault Is Organised

The vault follows the PARA + Zettelkasten hybrid structure:

- **00-inbox** — Unprocessed captures, fleeting thoughts, meeting notes
- **10-zettelkasten** — Permanent, atomic concept notes
- **20-projects** — Active, bounded projects with a clear finish line
- **30-literature** — Notes on books, papers, articles
- **40-teachings** — Dharma teachings, practice instructions, reflections
- **60-journals** — Daily notes and longer personal writing
- **70-sources** — Web clips, PDFs, ingested reference material

## The Core Habit

Capture everything in **00-inbox**. Process weekly: each fleeting note becomes either a permanent zettel (move to **10-zettelkasten**) or is discarded. Permanent notes should contain exactly one idea, stated clearly enough that your future self can understand it without the original context.
