# Atomic Notes

An atomic note expresses exactly one idea, completely enough to stand alone. The word *atomic* comes from the Greek *atomos* — indivisible. You should not be able to split the note further without losing coherence.

## The Test for Atomicity

Ask: *If I removed everything after the first paragraph, would the note still make a complete point?* If yes, the first paragraph is the note. Move the rest to new notes and link them.

## Common Violations

- **The summary note** — a bullet list of facts about a topic with no single argument. Split into one note per insight.
- **The quote dump** — copied text with annotations. Distil to one insight in your own words; link to a literature note for the source.
- **The process note** — a how-to with five steps. Each step that represents a transferable idea deserves its own note.

## Why Atomicity Matters for Retrieval

A dense, multi-idea note can only match one query well. An atomic note matches every query that touches its one idea — and since it links to related notes, retrieval naturally surfaces the surrounding context.

This is why [[Zettelkasten Method]] scales: the search surface grows proportionally with the number of atomic notes, not with the volume of text.
